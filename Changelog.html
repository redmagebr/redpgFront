0.52.3

Updates to the Sheet System allows styles to be less sub-optimized with regards to multiple fields updating the same thing.

Loading a sheet or switching sheets should become somewhat faster now, provided the style utilizes these changes.

0.52.2

Closing the webm screen will stop playing the webm.

0.52.1

Webms are no longer muted by default! They will use the same Volume as background musics for now.

Later Webms will get the following personal options: muted, autoplay, loop, volume.

0.52.0

It is now possible to use "{" and "}" as markers in Roleplay and Story messages. This will print the text inside as Bold Italic.

Text inside { and } will also not be translated in language speak.
