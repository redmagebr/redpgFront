0.52.1

Webms are no longer muted by default! They will use the same Volume as background musics for now.

Later Webms will get the following personal options: muted, autoplay, loop, volume.

0.52.0

It is now possible to use "{" and "}" as markers in Roleplay and Story messages. This will print the text inside as Bold Italic.

Text inside { and } will also not be translated in language speak.

0.51.0

New slash command: "/webm". Usage: "/webm url".

This will use the current Picture UI to play an HTML5 video (webm, mp4 or ogg/video).

The video will autoplay and loop, but it will be muted.

It is possible to draw over the video.

0.50.11

Made the Tables section aware of login changes.

Messages should no longer be lost when pressing Up and Down arrows while typing in the chat. Messages being written will always be added to the bottom of the scroll up/down list.

0.50.10

8081 port added as an option for the websocket ports. With regards to which port is best: 80 > 8080 > 8081. However, the one which works is the best, so try around if the chat can't connect.
