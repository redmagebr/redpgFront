0.41.4

Images in the variable "image" list are now tagged by Folder, making it easier to read.

0.41.3

Variable "image" now can have "data-img" attribute. data-img of "1" makes it print the image itself when vieweing.

Variable "image" will now auto-update it's list when the ImageDB is updated, not requiring you to reload the sheet.

Dragon Fantasy Saga 3.0 sheet now uses this kind of images for the avatar. So you need to update to use it without problems.

0.41.2

Icons to drawings buttons added.

Picture UI will attempt to remember all drawings seen.

Picture UI now differentiates between drawings of different rooms.

Picture UI now doesn't let one draw without a room open. Picture UI doesn't show drawings if there's no room open.

0.41.1

Sheets can use variables of type "sheet".

It is now possible to lock and unlock picture drawings. Only the storyteller can do so.

0.41.0

Longtext variables can now create more than one Paragraph through data-paragraph = 1.

New sheet: Generic Sheet. This sheet begins having no fields at all and you get to add them one by one.

0.40.7

Makes streaming borders smaller. Makes streaming scroll smaller.

0.40.6

Fixes some issues with streaming.

0.40.4

Some random fixes to browser history.

Attempted to fix Chrome randomly placing scroll outside of windows.

0.40.3

Following the standard of setting english as the default language for whichever parts will only exist in a single language, RedPG will no longer look for offline sound files in the "Sons" folder.

From now on, RedPG will look for offline sounds in the "Sounds" folder. So you need to rename it from "Sons" to "Sounds".

0.40.2

It is now possible to navigate through Browser History.

Hitbox chat now follows Styled Windows. Hitbox buttons are now smarter.

Video player now follows Styled Windows. Also works better in streaming mode.

0.39.3

Drawing now behaves like a brush, rather than a thousand lines.
